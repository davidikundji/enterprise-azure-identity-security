# PowerShell Automation Scripts

## Overview
This folder contains PowerShell scripts used to automate identity management tasks in Microsoft Entra ID.

These scripts demonstrate automation, scalability, and administrative efficiency in a cloud identity environment.

---

## Scripts Included

### create-users.ps1
Creates multiple users in Microsoft Entra ID using Microsoft Graph.

**Key Features:**
- Automates user provisioning  
- Supports bulk user creation  
- Reduces manual administrative effort  

---

### disable-inactive-users.ps1
Identifies and reviews active user accounts.

**Key Features:**
- Helps monitor account activity  
- Supports identity lifecycle management  
- Can be extended for automated account disabling  

---

### identity-report.ps1
Generates a report of all users in Microsoft Entra ID.

**Key Features:**
- Exports user data to CSV  
- Supports auditing and compliance  
- Improves visibility into identity environment  

---

- PowerShell
