# Lab 7 – Microsoft Defender for Cloud

## Overview

This lab demonstrates how to use Microsoft Defender for Cloud to assess, monitor, and improve the security posture of Azure resources. It focuses on identifying vulnerabilities, applying security recommendations, and monitoring threats.

---

## Objectives

- Enable Microsoft Defender for Cloud
- Assess security posture using Secure Score
- Identify and remediate security recommendations
- Monitor alerts and threats
- Understand cloud security posture management (CSPM)

---

## Technologies Used

- Microsoft Azure
- Microsoft Defender for Cloud
- Azure Virtual Machines
- Security Center

---

## Implementation Steps

### Step 1 – Enable Defender for Cloud

- Navigate to Microsoft Defender for Cloud
- Enable Defender plans for:
  - Virtual Machines
  - Storage Accounts

---

### Step 2 – Review Secure Score

- Check Secure Score dashboard
- Identify improvement areas
- Analyze recommendations

---

### Step 3 – Remediate Security Recommendations

Examples:
- Enable Just-in-Time VM access
- Apply system updates
- Enable disk encryption
- Restrict open ports

---

### Step 4 – Monitor Security Alerts

- Navigate to Security Alerts
- Review detected threats
- Analyze alert severity and recommendations

---

### Step 5 – Improve Security Posture

- Apply recommended fixes
- Track improvements in Secure Score
- Document changes

---

## Security Concepts Demonstrated

- Cloud Security Posture Management (CSPM)
- Threat detection and monitoring
- Vulnerability assessment
- Security hardening
